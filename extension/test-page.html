🧪 ScreenSense Extension Test Page

Use this page to test the extension and debug issues.

Test 1: Content Script Loaded?

Checks if the content script was injected successfully.

Test 2: FloatingPanel Class Available?

Checks if the FloatingPanel class was loaded.

Test 3: Recording Functions Available?

Checks if window.__screensense interface is available.

Test 4: Manually Show Panel

Directly calls the function to show the floating panel.

Test 5: Device Enumeration

Lists available cameras and microphones.

📋 Testing Instructions

First, reload the extension in chrome://extensions
Then refresh THIS page (F5)
Run each test above in order
If any test fails, note the error message
Try clicking the extension icon - the panel should appear on THIS page without redirect

✅ What Should Happen

When you click the extension icon on THIS page:

  1. Floating panel appears on the RIGHT side of this page
  2. Panel shows camera and microphone dropdowns
  3. When you click "Start Recording", screen share dialog appears
  4. After selecting screen, recording starts ON THIS PAGE (no redirect!)
  5. Camera overlay appears in bottom-right (if camera enabled)
  6. Control bar appears at top

NO REDIRECT should happen!
